# 接口文档说明



通用签名说明

```
每个接口必传3个参数
time     接口传参  时间戳
mark     接口传参  平台描述 自定义字符串建议：平台名称拼音全拼
sign	 接口传参  加密串。

加密方式 sign = md5($time.$mark.$key);

key      平台互相约定平台密钥 （找审核平台开发者索要密钥）

```

```入参参数 无特殊说明  均为字符串 string```

接口返回参数说明

```php
返回值实例
{
    "code":'1',  // 状态码，1成功，0失败。
    "msg":'成功', // 返回结果描述。 
    "data":[]    // 返回值，所有数据都包在data，具体格式要求看具体接口说明。
}

以上返回值，code与msg参数是每个接口固定返回参数，data根据接口情况而定是否返回。
   
```

## 1，审核平台修改活动名称 通知

<a id=修改活动标题> </a>

### 基本信息

**Path：** /vote/promote/replace_title

**Method：** POST

**接口描述：**  审核平台修改活动名称，通知相关平台活动信息修改。



### 请求参数

| 参数名称 | 是否必须 | 示例 | 备注     |
| -------- | -------- | ---- | -------- |
| catid    | 是       | 1486 | 活动id   |
| title    | 是       | 标题 | 新标题   |
| sign     | 是       |      | 签名     |
| time     | 是       |      | 时间戳   |
| mark     | 是       |      | 公共参数 |

### 返回参数

```json
{
    "code":1,  //0失败  1成功
    "msg":"成功"
}
```
